Some Terminology

Repository

Repository - Repositories are where you store each of your different projects. For example you might have one Repository called Website which contains a Website you are working on. You might also have another called Python Programs - containing (unsurprisingly!) All of your Python Programs.

Repositories can be created locally (on your computer) and then synchronised with GitHub so that you can work on them elsewhere by cloning and others can do this too

The Third Form course is optional, and builds on the basics taught in the First Form. Pupils create a website using HTML and CSS, learn to write programmes to solve simple problems using Python, investigate the world of digital marketing via social media, disassemble a computer to see what is inside, and explore the realm of physical computing using a Raspberry Pi.

GCSE

At GCSE we follow the OCR Computing syllabus. The course introduces pupils to the fundamentals of computer programming, teaching logic, and problem-solving skills. Whilst there is a sizeable practical content, there is a requirement to learn applications of computing in industry, business and commerce.

A Level

At A level the OCR course follows a natural progression from GCSE, and focuses on system production to solve problems using a programming language, and also looks at the architecture of computers, algorithms, and health and safety.
